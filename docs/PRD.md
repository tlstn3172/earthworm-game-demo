# Product Requirements Document (PRD)
# Snake Reborn - 모바일 웹 게임

## 1. 프로젝트 개요

### 1.1 제품 비전
**Snake Reborn**은 클래식 스네이크 게임을 현대적으로 재해석한 모바일 최적화 웹 게임입니다. 네온 그린 컬러와 미니멀한 디자인을 통해 세련되고 몰입감 있는 게임 경험을 제공합니다.

### 1.2 목표
- 직관적이고 반응성 높은 모바일 게임 경험 제공
- 라이트/다크 모드를 지원하는 현대적인 UI/UX
- 부드러운 애니메이션과 시각적 피드백으로 프리미엄 느낌 구현
- 오프라인에서도 플레이 가능한 PWA 지원

### 1.3 타겟 사용자
- 모바일 캐주얼 게임을 즐기는 사용자
- 클래식 스네이크 게임의 향수를 느끼고 싶은 사용자
- 짧은 시간에 즐길 수 있는 게임을 찾는 사용자

## 2. 기능 요구사항

### 2.1 핵심 게임 메커니즘

#### 2.1.1 게임 플레이
- **스네이크 이동**: 가상 조이스틱을 통한 4방향 이동 제어
- **먹이 시스템**: 
  - 먹이를 먹으면 스네이크 길이 증가
  - 점수 획득 (먹이당 10점)
  - 먹이는 랜덤 위치에 생성
- **장애물 시스템**:
  - 맵 내 고정 장애물 배치
  - 장애물 충돌 시 게임 오버
- **충돌 감지**:
  - 자기 몸과 충돌 시 게임 오버
  - 맵 경계 충돌 시 게임 오버
  - 장애물 충돌 시 게임 오버

#### 2.1.2 부스트 기능
- 우측 하단 부스트 버튼 (번개 아이콘)
- 일시적으로 이동 속도 증가
- 쿨다운 시간 적용

#### 2.1.3 점수 시스템
- 실시간 점수 표시
- 최고 점수 기록 및 표시
- 로컬 스토리지에 점수 저장

### 2.2 화면 구성

#### 2.2.1 시작 화면 (Start Screen)
**주요 요소:**
- 게임 로고 및 타이틀
  - "SNAKE REBORN" 텍스트
  - 네온 그린 강조 효과
  - 추상적인 3D 기하학적 배경 이미지
- 최고 점수 위젯
  - 트로피 아이콘
  - "BEST SCORE: 1,240" 형식
- 액션 버튼
  - **게임 시작** (Primary): 네온 그린 배경, 펄스 애니메이션
  - **설정** (Secondary): 투명 배경, 테두리 스타일
- 하단 툴바
  - 정보 버튼 (좌측)
  - 버전 정보 (중앙)
  - 사운드 토글 (우측)

**디자인 특징:**
- 그리드 패턴 배경 (40px × 40px)
- 네온 그린 글로우 효과
- 다크/라이트 모드 지원

#### 2.2.2 게임 플레이 화면 (Game Screen)
**상단 HUD:**
- 점수 카드
  - 트로피 아이콘
  - 현재 점수 (예: 1,250)
- 타이머 카드
  - 시계 아이콘
  - 플레이 시간 (MM:SS 형식)
- 일시정지 버튼

**게임 영역:**
- 그리드 패턴 배경
- 스네이크
  - 머리: 네온 그린, 눈 표현
  - 몸통: 점진적 투명도 (90% → 50%)
  - 흰색 테두리로 세그먼트 구분
- 먹이
  - 빨간색 원형
  - 영양 아이콘
  - 펄스 애니메이션
  - 빨간 글로우 효과
- 장애물
  - 회색 직사각형
  - 그림자 효과

**하단 컨트롤:**
- 가상 조이스틱 (좌측)
  - 반투명 원형 베이스
  - 방향 화살표 가이드
  - 네온 그린 썸
- 부스트 버튼 (우측)
  - 번개 아이콘
  - 네온 그린 원형 버튼
  - 활성화 시 애니메이션

#### 2.2.3 게임 오버 화면 (Game Over Screen)
**주요 요소:**
- 게임 오버 타이틀
  - 슬픈 이모티콘 아이콘
  - "Game Over!" 텍스트
  - 서브 텍스트: "Better luck next time, snake charmer."
- 점수 카드
  - 상단 네온 그린 액센트 바
  - "Final Score" 레이블
  - 최종 점수 (큰 폰트)
  - 최고 점수 비교 배지
- 액션 버튼
  - **다시 시작** (Primary): 회전 아이콘, 네온 그린
  - **메인 메뉴** (Secondary): 홈 아이콘
  - **점수 공유** (Secondary): 공유 아이콘

#### 2.2.4 설정 화면 (Settings Screen)
**헤더:**
- 뒤로가기 버튼
- "설정" 타이틀

**사운드 섹션:**
- 효과음 토글
  - 볼륨 아이콘
  - ON/OFF 스위치
- 배경음악 토글
  - 음악 아이콘
  - ON/OFF 스위치

**게임 플레이 섹션:**
- 조작 감도 슬라이더
  - 범위: 1-100%
  - 현재 값 표시
  - "느리게" ~ "빠르게" 레이블
- 난이도 선택
  - 3단계: 쉬움 / 보통 / 어려움
  - 버튼 그룹 형식
  - 선택된 항목 강조
- 진동 토글
  - 진동 아이콘
  - ON/OFF 스위치

**하단:**
- 설정 저장 버튼 (Primary)
- 버전 정보

### 2.3 디자인 시스템

#### 2.3.1 컬러 팔레트
```css
/* Primary Colors */
--primary: #7bf425           /* 네온 그린 */
--primary-dark: #5ec315      /* 다크 그린 */

/* Background */
--background-light: #f7f8f5  /* 라이트 배경 */
--background-dark: #172210   /* 다크 배경 */

/* Surface */
--surface-light: #ecf4e7     /* 라이트 서피스 */
--surface-dark: #24331a      /* 다크 서피스 */

/* Text */
--text-main-light: #131c0d   /* 라이트 모드 텍스트 */
--text-main-dark: #f0fdf4    /* 다크 모드 텍스트 */
```

#### 2.3.2 타이포그래피
- **Primary Font**: Spline Sans
- **Weights**: 300, 400, 500, 600, 700
- **크기 시스템**:
  - 타이틀: 3xl-4xl (36-48px)
  - 점수: 2xl-6xl (24-60px)
  - 본문: base-lg (16-18px)
  - 캡션: xs-sm (12-14px)

#### 2.3.3 아이콘
- **라이브러리**: Material Symbols Outlined
- **주요 아이콘**:
  - `cruelty_free`: 스네이크 로고
  - `emoji_events`: 트로피
  - `play_arrow`: 게임 시작
  - `settings`: 설정
  - `pause`: 일시정지
  - `replay`: 재시작
  - `bolt`: 부스트
  - `nutrition`: 먹이

#### 2.3.4 Border Radius
- Default: 0.25rem (4px)
- lg: 0.5rem (8px)
- xl: 0.75rem (12px)
- 2xl: 1rem (16px)
- full: 9999px (완전한 원형)

#### 2.3.5 그림자 효과
```css
/* Primary Glow */
box-shadow: 0 0 15px rgba(123, 244, 37, 0.5);

/* Button Shadow */
box-shadow: 0 4px 14px 0 rgba(123, 244, 37, 0.39);

/* Red Glow (Food) */
box-shadow: 0 0 15px rgba(239, 68, 68, 0.5);
```

#### 2.3.6 애니메이션
- **Pulse Animation**: 3초 주기, 부드러운 스케일 변화
- **Hover Effects**: 
  - 버튼: -translate-y-1, 그림자 증가
  - 카드: scale(1.02)
- **Active States**: scale(0.95-0.98)
- **Transitions**: 200-500ms duration

### 2.4 반응형 디자인
- **최대 너비**: 448px (max-w-md)
- **최소 높이**: max(884px, 100dvh)
- **Safe Area**: 모바일 노치 대응
- **터치 타겟**: 최소 44×44px

## 3. 기술 요구사항

### 3.1 프론트엔드 스택

- **프레임레이트**: 60 FPS

## 4. 사용자 경험 (UX) 요구사항

### 4.1 사용자 플로우
```
시작 화면
  ├─ [게임 시작] → 게임 플레이 화면 → 게임 오버 화면
  │                     ├─ [일시정지] → 일시정지 모달
  │                     └─ [게임 오버] → 게임 오버 화면
  │                                        ├─ [다시 시작] → 게임 플레이 화면
  │                                        ├─ [메인 메뉴] → 시작 화면
  │                                        └─ [점수 공유] → 공유 모달
  └─ [설정] → 설정 화면 → [뒤로가기] → 시작 화면
```

### 4.2 인터랙션 디자인
- **피드백**:
  - 먹이 획득: 햅틱 피드백 + 사운드
  - 충돌: 강한 햅틱 + 경고음
  - 버튼 클릭: 스케일 애니메이션
- **상태 표시**:
  - 로딩: 스피너 또는 스켈레톤
  - 에러: 토스트 메시지
  - 성공: 체크 아이콘 애니메이션

### 4.3 접근성 (Accessibility)
- **키보드 네비게이션**: Tab, Enter, Arrow keys
- **ARIA 레이블**: 모든 인터랙티브 요소
- **색상 대비**: WCAG AA 기준 준수
- **포커스 인디케이터**: 명확한 아웃라인

## 8. 성공 지표 (KPI)

### 8.1 사용자 참여

- **평균 세션 시간**: 목표 5분
